# YT Downloader
## Simple Desktop app to download Youtube videos


### Setup

```
    # for linux
    chmod +x ./scripts/run.sh
    ./scripts/run.sh

    # for windows
    ./scripts/run
```


### Build Executable

```
    # for linux
    chmod +x ./scripts/build.sh
    ./scripts/build.sh

    # for windows
    ./scripts/build
```
